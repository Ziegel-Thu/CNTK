# paper

One folder per paper or project draft.

For arXiv papers, keep source/TeX when available:

```text
paper/NNN-author-year-keyword/
├── README.md
├── paper.pdf
├── source/
└── notes.md
```

Download source from:

```text
https://arxiv.org/e-print/{arxiv_id}
```

Use `000-...` for local drafts.
